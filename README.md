# simplefuckingrichpresnce
A FiveM Rich presence script someone wanted me to make

# Installation
extract the compressed folder \n
make an application here: https://discord.com/developers/applications \n
add an image to Rich Presence Art Assets \n
put the client id of the application in the client.lua where it says CLIENT_ID_HERE \n
put the image name fromm Rich Presence Art Assets in the client.lua where it says IMAGE_NAME_HERE \n
put discord link or whatever other text you want where it says DISCORD_LINK_HERE \n
put it in resources \n
add it to server.cfg \n
restart server or do `refresh` in console then do `start (resource name` \n
have fun ig \n
